# Pure JavaScript Table Search

A complete search in a table

## Example

```
<input id='search' type='text' onkeyup='searchTable(this,"data","tr","td")'>

<table id='data'>
  <tr>
    <td>Potato</td>
    <td>Brown</td>
  </tr>
  <tr>
    <td>Paprika/td>
    <td>Green</td>
  </tr>
  <tr>
    <td>Banana</td>
    <td>Yellow</td>
  </tr>
  <tr>
    <td>Apple</td>
    <td>Red</td>
  </tr>
</table>

<script src="tableSearch.js"></script>
```
